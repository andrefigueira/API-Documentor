cFields 1.0

A lightweight plugin that allows you to have more control over the design of Selects and checkboxes.

Usage

Using cFields is easy, all you do is include the cFields file, add the CSS to yours, the add the function call on your javascript file or in your head, you can use it to highly customise selects and checkboxes.

$(document).ready(function(){
	
	$('#network').cFields({label:true});
	
});
